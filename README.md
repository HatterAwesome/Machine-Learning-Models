# StreetHouseDataset
  1.Using TensorFlow, I modified the original tutorial to use the Street View House Numbers (SVHN) dataset instead of MNIST. The key change involved replacing 28×28 grayscale images with 32×32 RGB images, requiring updates to both the generator and discriminator architectures to handle three color channels and larger image dimensions. The generator was adjusted to output 32×32×3 images, while the discriminator was modified to accept this new input shape. To improve training speed in Google Colab, the dataset was reduced to 10,000 images, the number of epochs was lowered to 5, and the batch size was set to 64. During training, generated images were saved after each epoch and displayed in a 4×4 grid format. The results show that the model begins to learn structural patterns from the dataset, but the generated images remain blurry and not fully recognizable due to limited training time and reduced dataset size. Overall, this project demonstrates how GANs can be adapted to real-world datasets and highlights the importance of sufficient data and training duration for producing high-quality generated images.

# Vectors 
  1. This project demonstrates Natural Language Processing (NLP) vectorization techniques, focusing on transforming text into numerical representations that machine learning models can understand. Traditional approaches such as One-Hot Encoding, Bag of Words (BoW), and TF-IDF are implemented to illustrate basic text representation methods, while pre-trained GloVe word embeddings are used to showcase more advanced, dense vector representations. The embeddings capture semantic relationships between words, allowing similar words (e.g., “cat” and “dog”) to have closer vector representations in high-dimensional space. This implementation highlights the difference between sparse and dense representations and demonstrates how unsupervised learning techniques can be used to extract meaningful patterns from text data without labeled outputs.

# Stable Diffusion
  1. This project implements a text-to-image generation model using Stable Diffusion in Google Colab. Stable Diffusion is a generative diffusion model, generally considered unsupervised or self-supervised learning, that creates images from text prompts. The code uses several libraries including PyTorch for deep learning, Diffusers to access the Stable Diffusion pipeline, Transformers to process the text prompt into embeddings, Accelerate to improve GPU performance, Safetensors to safely load model weights, Scipy for computational support, and Pillow for image handling. The model is loaded using a pre-trained version and run on the GPU for efficiency. A descriptive text prompt is provided, and the model generates an image by starting with random noise and gradually removing it over multiple steps—in this case 25 iterations—until a clean image matching the prompt is produced. The final image is displayed and saved as output.png. Note that GitHub may show a preview error for the notebook due to metadata issues, but the code and results are fully functional.
